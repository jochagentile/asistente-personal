API de inteligencia artificial
Para usar el campo de lenguaje natural necesitás una API key de Anthropic. Es gratuita para empezar y muy económica.

1. Entrá a console.anthropic.com
2. Creá una cuenta → API Keys → Create Key
3. Pegá la key acá abajo. Se guarda solo en este dispositivo.
